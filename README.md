# SyncAvatar for VRChat

A desktop application for viewing, editing, and syncing VRChat avatar parameters.

## Features

- View all editable avatar parameters in one interface.
- Modify parameters without opening the in-game Expressions menu.
- Optionally sync selected parameters with multiple friends in real time.
- Simple graphical interface with no command-line setup required.

you can use it local only or to sync with friends as well!

## Download

1. Download `SyncAvatar.exe` from the [**Releases**](https://github.com/RowsOfCrows/SyncAvatarVRC/releases) page.
2. Run the executable.
3. On first launch, a `config.ini` file will be created automatically.
4. Edit `config.ini` if you want to sync with friends.

## About

SyncAvatar started as a command-line utility for synchronizing avatar parameters between two users. Later, I built a separate GUI that displayed and edited avatar parameters outside of VRChat. Eventually, I combined both projects into a single application, bringing parameter management and synchronization together in one place.
